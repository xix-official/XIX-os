\# XIX Code of Conduct (2026)



This project is part of the XIX ecosystem.

To maintain a healthy, safe and productive community, all participants must follow this Code of Conduct.



\---



\## 1. Expected behavior



Collaborators must:



\- Treat everyone with respect.

\- Maintain a professional environment.

\- Avoid insults, personal attacks or toxic behavior.

\- Communicate ideas clearly and constructively.

\- Keep discussions focused on the technical.

\- Assume good intention on the part of other collaborators.



\---



\## 2. Disallowed behavior



Will not be tolerated:



\- Insults, threats, harassment or discrimination.

\- Trolling, spam or disruptive behavior.

\- Publish private information of others.

\- Start unnecessary conflicts or off-topic discussions.

\- Send malicious code, backdoors, viruses or hidden telemetry.

\- Try to harm the project or its users.



Any violation may result in warnings or expulsion from the project.



\---



\## 3. Contributions



By contributing to XIX, you agree:



\- Follow this Code of Conduct.

\- Send only original code or compatible license.

\- Do not include binaries, malware, obfuscated code or destructive scripts.

\- Maintain code quality and respect the project structure.

\- Ensure that your contribution does not break the build or eliminate essential functions.

\- Clearly explain changes in large PRs.

\- Open a discussion before major changes.



\---



\## 4. Report problems



If you see a violation of the Code of Conduct:



\- Report it privately to maintainers.

\- Include details and evidence if possible.

\- Don't start public fights or personal attacks.



Maintainers will review the situation and act accordingly.



\---



\## 5. Responsibility of maintainers



Maintainers must:



\- Enforce this Code of Conduct fairly.

\- Review contributions with respect.

\- Protect the project from toxic or malicious behavior.

\- Maintain a safe and productive environment.



\---



\## 6. Scope



This Code of Conduct applies to:



\- Issues

\- Pull Requests

\- Discussions

\- Chats

\- Documentation

\- Any official space of the XIX community



\---



\## 7. Commitment



By participating in the XIX project, you agree to comply with this Code of Conduct to maintain a healthy and professional environment.
